# CPU Scheduling Simulator

A Python-based Operating System project that simulates CPU scheduling algorithms with GUI visualization and performance comparison.

## Features
- FCFS
- SJF
- Priority Scheduling
- Round Robin
- Gantt Charts
- Performance Metrics

## Technologies Used
- Python
- Tkinter
- Matplotlib

## Learning Outcomes
- Process scheduling
- CPU optimization
- Algorithm comparison
